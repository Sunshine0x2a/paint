# 一个简单的图形操作界面：
---
## figure类设计及目的：
目的： 选择自建类：锻炼自己的编程能力
### 设计：
+ 图形的生命周期交给command类,并使用shared_ptr进行管理
+ contain方法，共有三个可能返回值，0代表不在，1代表在粗略碰撞箱内，2代表在精准碰撞箱内
## ViewTransform类：负责转化全局坐标与视图坐标
## 功能实现原理：
### 调整图形大小 
目前只允许实现单个图形的大小调整
### 关于复制与粘贴的实现：
当复制多个对象时，将其打包为一个复合图形指针，进行存储，当粘贴时在由PasteCommand进行解压与复制
### 菜单项：
normalMenu:
添加->图形菜单
粘贴
撤销
重做
恢复默认比例

selMeun:
复制
删除
组合
拆散
